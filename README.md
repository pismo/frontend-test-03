# Abraão Bueno

> Requisitos: **NodeJs** e **npm**

### Dependencias:

```npm install```

### Run (*development mode*)

```npm run dev```

O navegador abrirá automáticamente, caso não abra, entre com a url: localhost:9000

### Run (*production mode*)

```npm run prod```

O desenvolvimento foi feito utilizando:

- React
- Redux
- React-router
- Material-ui
- Styled-components
